Content Allowed in Mapplet Info Windows

Note: The Google Mapplets API has been officially deprecated as of May 19, 2010. Existing Mapplets will continute to be supported, but no new Mapplets will be accepted into the Mapplets Gallery.

Since the HTML placed into info windows runs within the Google Maps domain, we must santize its contents to prevent any malicious code from executing. No Javascript is allowed and only the tags and CSS listed on this page are allowed.
