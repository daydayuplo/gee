Sample Search Plug-ins

To get you started with the Google Earth Enterprise Search Framework API, Google provides some sample plug-ins with the Google Earth Enterprise system. They are located in /opt/google/search/plugins.

CoordinatePlugin 

<?xml version="1.0" encoding="UTF-8" standalone="no" ?>
<SearchTabDefinition>
<label>Coordinates</label>
<fields>
 <field>
 <label>Latitude, Longitude</label>
 <key>location</key>
 </field>
 </fields>
<url></url>
<plugin_name>CoordinatePlugin</plugin_name>
<query_params>flyToFirstElement=true&displayKeys=location</query_params>
<is_poi>0</is_poi>
</SearchTabDefinition>
